# Type of Systems

## OLTP (Online Transaction Processing) -> RDBMS, Document Stores, KV Stores

- Ingestion (over time)
  - Day-to-day activities

- Work with
  - Individual Records

## OLAP (Online Analytical Processing) -> DataWarehouse / DataLake

- Ingestion (batched or streamed)
  - Data from OLTP

- Work with
  - Tons of records together
    - Analysis
    - Machine Learning
