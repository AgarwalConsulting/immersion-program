# Batch vs Stream Processing

## Ingestion

- Delivery of Data (Stream Processing)

  AWS: MSK, SNS (Simple Notification Service), EC2, ...

## Processing

- Batch Processing

  AWS: RedShift, EMR (Elastic Map Reduce), EC2, Lambda, ...

- Stream Processing

  - AWS: Kinesis

## Storage

  AWS: RDS, S3, Aurora, Athena, RedShift, EBS, ...
