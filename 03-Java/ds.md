# Data Structure

## Array - Statically sized - contiguous memory

```java
bool b[] = new bool[3];
int a[] = new int[3];

// Accessing or assign

a[0] = 10;
System.out.println(a[1]);
```

## ArrayList - Dynamically sized - fragmented memory

- Derived Types only

```java
List<Integer> a = new ArrayList<Integer>();

// a = Arrays.asList(1, 2, 3, 5, 7);

// Accessing or assign
a.get(0);
a.add(1);
```
