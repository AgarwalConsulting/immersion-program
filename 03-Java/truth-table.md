# Truth table

## AND (`&&`)

```
x          y              Result
--------------------------------
false     false           false
false     true            false
true      false           false
true      true            true
```

## OR (`||`)

```
x          y              Result
--------------------------------
false     false           false
false     true            true
true      false           true
true      true            true
```

## NOT (`!`)

```
x         Result
----------------
false      true
true       false
```
