# SQL vs NoSQL

- SQL - Structured Query Language
  - Relational Algebra
    - Joins
  - Set Theory
  - Eg. AccessDB, Sqlite, Oracle, Postgres, MySQL, SQLServer, ...

- NoSQL
  - Unstructured DBs
  - Schemaless
  - No SQL support
    - Requires a DB specific to work with
  - Eg. MongoDB, BigTable, Neo4J, ...
